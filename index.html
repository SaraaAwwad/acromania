Acromania


How To Play

Acromania is a chat room with a permanent word game running. The goal is to try to make a sentence with the random alphabets given as quick as possible.
Answers will be announced anonymously, you can vote to who you think came up with the best sentence.

Rules:
1- You need to give a valid answer that matches the random matching pattern (in case you play.)
2- You need to give a valid vote that matches any number in the list of answers (in case you vote.)
3- Have fun! ⚡
